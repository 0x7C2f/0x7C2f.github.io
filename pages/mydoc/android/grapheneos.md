## GrapheneOS
GrapheneOS is an Android based open-source operating system that has a primary focus on security and privacy (although more so on the security side than privacy).

### Project Links
- [Website](https://grapheneos.org/)
- [Source code](https://github.com/GrapheneOS)

### Pros
- Completely sandboxed Google Play Services.
- Storage scopes
- Increase in user profile count
- Pin scrambling
- Network/Sensor toggling
<br>
<br>
[and more...](https://grapheneos.org/features)

### Cons
- Partially degraded performace ([see here..](https://y.com.sb/watch?v=7iS4leau088))
- Various claims about toxic community
